Projects

Most Played

The most played app uses Spotify's web api to show the user their most played songs over three possible periods of time. Another cool feature is that a sample of each song will play when the user moves their mouse over it. I created this because I love Spotify's year in review where they show you your most played songs, this app allows you to look whenever you want. I created this in React. For backend authentication, the app links to a node app provided by Spotify developers.

Scoreboard

A scoreboard for American Football made using react. Users can add different types of score integers to their teams overall score (touchdown, field goal, safety, two point conversion) and subtract points if needed.

Name Shuffler

An app I created using React which takes names from user input and allows the user to shuffle their order. I created this after my friends and I couldn't find a tool to randomize the order of presenters for a Zoom "power point party" that didn't ask for our email address. The equivalent of drawing names from a hat but optimized for a global pandemic.

City Weather

An app that uses the OpenWeather API to display the current weather in a user-chosen city. Coded using vanilla JS.

Presidential Quotes

A simple trivia game made using vanilla Javascript where the user guesses which U.S. President said a certain quote.

Recipe Book

A "recipe book" that allows the user to store recipes including ingredients and instructions. It utilizes local storage to save data. Made using HTML, CSS, and vanilla Javascript
